
# Arq-web-api


# Variables de entorno 
Crear un archivo .env con las siguientes variables :  

DB_USER=postgres  

DB_PASSWORD=admin  

DB_HOST=postgres  

DB_PORT=5432  

DB_DATABASE=reuniones  

DB_DIALECT=postgres  




### Uso  


Instalacion de dependencias :  

`npm install`  


`npm start`
